# 更新日志

## 0.4.0 - 2026-06-19

### 新增

- 菜单编辑器升级为轻量运营台：历史版本、撤销/重做、批量操作、固定紧凑布局与原比例预览。
- 背景图新增本地资产库能力，按 hash 去重，支持内联资产包导入/导出和旧 Data URL 背景兼容；Page 不再单独展示资产中心入口。
- 背景编辑模式支持锁定/解锁，提供缩放、X/Y、居中、铺满、适应、重置、遮罩、模糊、亮度设置；大图编辑先使用临时对象 URL 预览。
- 样式字段扩展字体、卡片间距、分组内边距、阴影强度、边框强度、背景遮罩、背景模糊、背景亮度、水印文本。
- 聊天侧新增菜单别名、`/menu list`、`/menu search 关键词`、`/menu refresh [id]` 和按平台/群聊上下文默认菜单。
- 渲染状态接口追加排队时间、重试次数、指纹、缓存体积和队列长度，新增缓存与未使用资产清理能力。

### 修复

- Page 启动兼容性增强：移除 top-level await 和单一 module 入口依赖，增加页面桥接候选、`structuredClone`/`localStorage`/`replaceChildren` 兼容兜底，避免旧 WebView 中出现按钮全部失效。
- 所有顶部按钮、隐藏样式按钮、批量按钮、折叠按钮、主题选择、方案新建/复制/删除、菜单切换、预览层点击均增加异常兜底与回归审计。
- 重构新建、复制、删除、菜单切换状态流：未保存内容会先保留为本地草稿，快速连续新建/复制/删除不再卡死或导致方案选择器失效。
- 删除菜单不再依赖 WebView 原生 `confirm()`，改为 Page 内确认弹窗，避免点击删除后直接显示“已取消删除”。

### 调整

- 前端拆分为无打包原生脚本模块：运行时、状态、API、预览、弹窗、背景、校验、快捷键由独立文件承载，`app.js` 负责编排。
- 移除 Page 中的命令面板、资产中心、回收站、路由设置、UI 密度切换和多设备预览入口，减少按钮拥挤与误触。
- 导入前展示冲突预览；保存/删除前创建历史快照；删除菜单继续走后端安全删除流程。

## 0.3.0 - 2026-06-19

### 新增

- 编辑器增加“已修改 / 保存中 / 已保存”状态、离开/切换菜单确认、本地草稿自动保存与恢复/丢弃流程。
- 增加前端即时校验、保存前定位首个错误、分组/菜单项折叠记忆、菜单项搜索过滤。
- 主题预设升级为可预览卡片，支持复制当前样式到其他菜单、一键重置样式。
- 背景设置增加缩放滑杆、X/Y 数字输入、居中、铺满、重置背景，数字控件与拖动裁剪实时同步。
- 编辑栏布局压缩优化，移除菜单模板新建入口，减少按钮拥挤与误触。
- 增加智能/自定义分组间距设置，自定义范围放宽到 `0-200`（`0` 为完全贴合），Page 预览、浏览器渲染与 Pillow 回退渲染保持一致。
- 修复菜单项小卡片折叠按钮在部分 Page iframe 环境中点击后不生效的问题。
- 新增只读接口 `GET /astrbot_plugin_bot_menu/menus/render-status/<menu_id>`，Page 保存后展示渲染缓存状态。

### 调整

- 将后台缓存调度逻辑整理到 `MenuRenderCoordinator`，`main.py` 只负责调用保存后预渲染、聊天侧读取缓存和 Web API 编排。
- 上传大图时仅提示可能影响保存和加载速度，不限制上传尺寸。

## 0.2.0 - 2026-06-19

### 新增

- 支持上传自定义背景图，上传尺寸不受限制。
- 支持在 Page 实时预览中拖动背景位置，并通过边框四角调整缩放和裁剪。
- 支持前景菜单透明度 `0%` 到 `100%`，覆盖 Page 预览、浏览器渲染与 Pillow 降级渲染。
- 新增插件图标 `logo.png`，用于 AstrBot 插件列表展示。
- 保存菜单后自动在后台渲染并缓存菜单图片，聊天指令优先直接发送缓存图片。

### 修复

- 保存并重新打开编辑器后保留背景裁剪位置，不再自动对齐到顶部。
- 恢复预览区域独立滚动，长内容编辑时左侧预览与右侧表单互不遮挡。
- Pillow 降级渲染现在遵循智能宽度、每行卡片数、横幅卡片和前景透明度。
- 重新保存菜单后会替换旧缓存，避免指令触发时重复渲染导致响应变慢。

### 发布准备

- 统一版本号为 `0.2.0`。
- 作者更新为 `雪碧bir`。
- 更新 README 与发布说明，补齐安装、背景裁剪、透明度和渲染说明。

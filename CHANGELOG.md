# 更新日志

## 0.2.0 - 2026-06-19

### 新增

- 支持上传自定义背景图，上传尺寸不受限制。
- 支持在 Page 实时预览中拖动背景位置，并通过边框四角调整缩放和裁剪。
- 支持前景菜单透明度 `0%` 到 `100%`，覆盖 Page 预览、浏览器渲染与 Pillow 降级渲染。
- 新增插件图标 `logo.png`，用于 AstrBot 插件列表展示。

### 修复

- 保存并重新打开编辑器后保留背景裁剪位置，不再自动对齐到顶部。
- 恢复预览区域独立滚动，长内容编辑时左侧预览与右侧表单互不遮挡。
- Pillow 降级渲染现在遵循智能宽度、每行卡片数、横幅卡片和前景透明度。

### 发布准备

- 统一版本号为 `0.2.0`。
- 作者更新为 `雪碧bir`。
- 更新 README 与发布说明，补齐安装、背景裁剪、透明度和渲染说明。
